# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Ejecutar para iniciar la app

```
npm run start 
npm run srart:dev
```

5. Tener instalado docker
6. Levantar la base de datos en docker
```
docker-compose up -d
```
